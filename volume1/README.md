# 《少年派的数学》上册
适用于七年级数学

视觉上要比抽象的内容更容易学习。因此，为了便于学生更容易接受，此书以初中几何内容作为起始章节。这一点北师大教材做的挺好。

第一章 几何的起步：点，线，面，角
1. 点
2. 线

    2. 相交，平行，平移
    3. 长短，和，差，中点
3. 面
    1. 图形：平面和立体
    2. 多边形（三角形）和圆（尺规作图）
    3. 平移，旋转，轴对称
4. 角

5. 平面直角坐标系
